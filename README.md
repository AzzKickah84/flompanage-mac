# Flompanage for macOS

Desktop app for moderating [Flompert](https://www.flompert.nl).

## Download

Get the latest DMG from [Releases](https://github.com/AzzKickah84/flompanage-mac/releases/latest).

- **Apple Silicon** — for M1/M2/M3 Macs
- **Intel** — for Intel Macs

Drag **Flompanage** to **Applications** to install.

Installed Flompanage checks this repo automatically for updates.

## Source code

This repository intentionally contains **no application source code** — only this README and release DMGs.  
Source is built locally or via temporary CI pushes that are removed after each publish.
