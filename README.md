# Flompanage for macOS

Builds and releases the macOS version of Flompanage (Flompert.TV admin app).

## Download

Get the latest DMG from [Releases](https://github.com/AzzKickah84/flompanage-mac/releases/latest).

## Build (maintainers)

Requires macOS with .NET 9 + MAUI workload. See `Flompanage/Mac/README.md`.

Or trigger **Actions → Build Flompanage Mac** on GitHub.
